# Influencer Analytics & Funnel Performance Dashboard

## Description
This project analyzes influencer performance data to identify top-performing influencers and understand their behavior throughout the marketing funnel. 

**Key features include:**
- **Data Aggregation:** Summarized metrics per influencer (impressions, clicks, add-to-cart, purchases, revenue).  
- **Funnel Metrics:** CTR, ATC Rate, Purchase Rate, Overall Conversion, RPC, RPI.  
- **Influencer Scoring:** Composite score based on weighted funnel metrics to rank influencers.  
- **Segmentation:** Classifies influencers into Low, Mid, and High Performers.  
- **Visualizations:** Pie chart of segment distribution, top 20 influencers by revenue, segment-wise bar plots, funnel flow, and heatmap.  
- **CSV Export:** Generates a summary report for dashboard integration or further analysis.  

**Technologies Used:** Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook / VSCode  

**Purpose:**  
This notebook serves as a portfolio project showcasing data analysis, performance scoring, segmentation, and visualization skills. It can be used to understand influencer marketing effectiveness and guide strategy decisions.

## Proje (Türkçe)

### Açıklama
Bu proje, influencer performans verilerini analiz ederek en iyi performans gösteren influencer’ları belirlemeyi ve pazarlama funnel’ındaki davranışlarını anlamayı amaçlar.

**Temel özellikler:**
- **Veri Toplama:** Her influencer için özet metrikler (impressions, clicks, add-to-cart, purchases, revenue).  
- **Funnel Metrikleri:** CTR, ATC Oranı, Satın Alma Oranı, Genel Dönüşüm, RPC, RPI.  
- **Influencer Skoru:** Funnel metriklerine ağırlık vererek influencer’ları sıralayan kompozit skor.  
- **Segmentasyon:** Influencer’ları Düşük, Orta ve Yüksek Performans gruplarına ayırma.  
- **Görselleştirmeler:** Segment dağılımı pasta grafiği, gelire göre ilk 20 influencer, segment bazlı bar plotlar, funnel flow ve heatmap.  
- **CSV Export:** Özet rapor oluşturma, dashboard entegrasyonu veya ileri analiz için.  

**Kullanılan Teknolojiler:** Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook / VSCode  

**Amaç:**  
Bu notebook, portföy projesi olarak veri analizi, performans skorlaması, segmentasyon ve görselleştirme becerilerini sergilemek için kullanılabilir. Influencer pazarlama etkinliğini anlamaya ve strateji kararlarını yönlendirmeye yardımcı olur.
